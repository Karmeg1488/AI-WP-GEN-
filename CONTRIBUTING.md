# Contributing to AI WP GEN

We welcome contributions to AI WP GEN! Here's how you can help:

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://gitlab.com/YOUR_USERNAME/ai-news-generator.git`
3. Create a feature branch: `git checkout -b feature/amazing-feature`
4. Make your changes
5. Commit your changes: `git commit -m 'Add amazing feature'`
6. Push to the branch: `git push origin feature/amazing-feature`
7. Create a Merge Request

## Code Style

- Follow WordPress coding standards
- Use proper sanitization and escaping
- Add nonce checks for all AJAX endpoints
- Comment your code with clear explanations

## Testing

Before submitting:
1. Test all features with different WordPress versions
2. Test with multiple languages
3. Check admin panel functionality
4. Verify API key handling and error responses

## Reporting Issues

- Check existing issues first
- Provide detailed description
- Include WordPress version, PHP version, and error logs
- Specify which language was used for testing
